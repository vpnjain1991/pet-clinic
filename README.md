# pet-clinic
Pet Clinic Application-

Set Up-

1. git clone git@github.com:vpnjain1991/pet-clinic.git

2. cd pet-clinic

3. ./mvnw clean install & ./mvnw spring-boot:run -pl pet-clinic-web
